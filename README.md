## Klasifikasi Teks Publikasi Medis menggunakan IndoBERT

Proyek ini bertujuan untuk mengklasifikasikan dokumen ilmiah biomedis (Thyroid, Colon, dan Lung Cancer) menggunakan model Deep Learning IndoBERT.

## 📁 Struktur Repositori
- `Teks_Klasifikasi_IndoBERT.ipynb`: File utama pemrosesan data dan pemodelan.

## 📊 Metodologi
1. **Data Collection**: Dataset dari Kaggle (Biomedical Text Publication).
2. **Preprocessing**: Pembersihan teks dan label encoding.
3. **EDA**: Visualisasi sebaran kelas dan Word Cloud.
4. **Modelling**: Fine-tuning model IndoBERT.
5. **Evaluation**: Akurasi, F1-Score, dan Confusion Matrix.

## 🚀 Hasil
Model berhasil mencapai performa yang stabil dengan tingkat akurasi tinggi. Detail evaluasi dapat dilihat pada notebook.
